# Sample Bank App with Golang
## これは何
- Go入門のため手始めに https://docs.microsoft.com/ja-jp/learn/modules/go-write-test-program/ に沿って実装した銀行アプリ
- 以下を学んだ
  - 変数の初期化
  - if文
  - for文
  - interfaceとimprement
  - パッケージの作成と呼び出し
  - net/httpパッケージ
  - testingパッケージを用いたテストコードの記述

## 動かし方
```bash
$ go run bankapi/main.go
```
